Guia de configuracion de One page

Bienvenido a la guia de configuracion de one page, donde se indicara la funcion de cada seccion de su pagina, asi como las productos y servicios que puede asociar.
Agregar una imagen de cabecera
  • Para agregar una imagen de cabecera presionar el boton agregar imagen, representado por
  • Seleccionar una imagen de su equipo.
Agregar una caracteristica
  • Este punto cuenta de 2 partes: agregar una imagen y agregar la descripcion
  • Para agregar una imagen de presionar el boton agregar imagen, representado por
  • Seleccionar una imagen de su equipo y subirla
  • Para agregar una descripcion, presionar el boton agregar descripcion, representado por
  • Ingresar la descripcion en el cuadro emergente y presionar guardar.
Agregar informacion de su negocio
  • Este punto cuenta de 2 partes: agregar una descripcion usando el boton y agregar una imagen usando el boton
Producto o servicio
  • Este punto cuenta de 2 partes: agregar una imagen y agregar la descripcion del producto o servicio
  • Para agregar una imagen debe presionar el boton agregar imagen, representado por
  • Seleccionar una imagen de su equipo y subirla
  • Para agregar una descripcion, presionar el boton agregar descripcion, representado por
  • Ingresar la descripcion en el cuadro emergente y presionar guardar.
